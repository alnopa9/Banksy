Welcome to the Banksy Project!

Our goals for this project are to:

  • Use XML to start a database and mark up his graffiti tags that he has done around the world
  • Use KML (keyhole markup language) to create an overlay that pinpoints each location that Banksy has left a tag.
  • Use the map to look for frequencies in his tagging by comparing where and how often does he tag in a country
  • Create a timeline that includes his tags, sculptures, exhibitions, paintings, and books
